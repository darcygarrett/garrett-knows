# Garrett Knows

A personal helper app for staying connected with your people.

## Features

- Track birthdays
- Email notifications to subscribers
- Multiple people can subscribe to the same birthday
- Prevents duplicate notifications per day

## Quick Start

```bash
npm install

# start the web server
node server.js
```

Then open http://localhost:3000 in your browser.

You can also run the demo script:
```bash
npm run dev
```
