# Garrett Knows

A personal helper app for staying connected with your people.

## Features

- Track birthdays
- Email notifications to subscribers
- Multiple people can subscribe to the same birthday
- Prevents duplicate notifications per day

## Quick Start

```bash
npm install

node demo.js
```
